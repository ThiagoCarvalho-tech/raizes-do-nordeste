# Raízes do Nordeste

Sistema web desenvolvido como MVP para a franquia fictícia **Raízes do Nordeste**.

## Funcionalidades

- Login de clientes
- Escolha da unidade
- Cardápio digital
- Carrinho de compras
- Pagamento simulado
- Histórico de pedidos
- Programa de fidelidade
- Promoções
- Interface responsiva

## Tecnologias

- HTML5
- CSS3
- JavaScript
- LocalStorage
